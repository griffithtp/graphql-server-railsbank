GraphQL Server demo - RailsBank API
===================================

1. register your apikey
2. create apikey-demo.json in the following format

```
{
  "api_url": "https://play.railsbank.com/",
  "role": "customer-admin",
  "api_key": "{your_apikey}"
}

```
3. `npm run dev`
4. `localhost:9000/graphql`
